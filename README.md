# 🤖 AI-Powered Chatbot (Llama 3)
Developed a real-time chatbot using **Python** and **Streamlit**, integrated with the **Llama 3.3-70b** model via the **Groq API**.

### 🌟 Key Features
- **Natural Language Processing**: High-speed AI responses using Groq's LPUs.
- **State Management**: Maintains chat history during the session.
- **Secure Integration**: Uses environment variables for API safety.
